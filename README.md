# Avisi TechLab Dask

## Vereisten
- Pipenv

## Gebruik
- Installeer de packages uit [Pipfile](./Pipfile) door het volgende commando uit te voeren: `pipenv install`
- Open JupyterLab door het volgende commando uit te voeren: `pipenv run jupyter lab`
